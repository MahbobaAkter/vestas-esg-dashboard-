
# 🌍 Vestas ESG Dashboard Prototype

## 📌 Project Overview
This project is a prototype ESG (Environmental, Social, Governance) dashboard website created as part of the AI course assignment.

The goal of the project is to present sustainability information of Vestas in a simple, clear, and visual way using a web interface.

---

## 🏢 About Vestas
Vestas is a global leader in sustainable energy solutions, specializing in wind energy. The company focuses on reducing carbon emissions, promoting renewable energy, and improving circularity and sustainability practices.

---

## 📊 Features
- ESG dashboard visualization
- Key sustainability indicators (KPI)
- Clean and simple web interface
- Prototype-level implementation

---

## 📈 Key ESG Indicators
- 🌱 ~37,000 tonnes CO₂ avoided  
- 🏗️ 22,000+ tonnes low-emission steel  
- ♻️ Zero-waste wind turbine goal by 2040  
- 👷 Strong focus on safety and workforce  

---

## 🧠 Insights
Vestas shows strong ESG performance, especially in environmental impact due to its renewable energy focus. The company is also improving in social responsibility and governance transparency.
